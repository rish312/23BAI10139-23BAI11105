State Representation Learning for Deep Reinforcement Learning 


Tanya Sahu 23BAI11105
Ekansh Sukla 23BAI10139


This project explores the integration of Representation Learning techniques with Deep Reinforcement Learning
(DRL) to investigate how learned state representations influence agent performance. We implement a Deep
Q-Network (DQN) agent and compare its performance using three approaches: (1) raw state observations, (2)
autoencoder-compressed representations, and (3) contrastive learned representations. Experiments are
conducted on OpenAI Gymnasium's CartPole-v1 and LunarLander-v2 environments. Our results provide insights
into when representation learning helps or hinders RL performance, and demonstrate the structural properties of
different learned latent spaces.
